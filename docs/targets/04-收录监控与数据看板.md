## 指标口径：每个工具看什么

### Search Console

每天看：

- Pages indexing。
- Sitemap status。
- Queries。
- Pages。
- Countries。
- Devices。

最重要指标：

- impressions。
- clicks。
- average position。
- CTR。
- indexed pages。

当前判断优先级：

1. 先看页面是否进入索引。
2. 再看是否有 impressions。
3. 最后才看排名和 CTR。

### GA4

重点指标：

- users。
- sessions。
- page views。
- engagement rate。
- average engagement time。
- landing page。
- source / medium。

当前最重要的是：

- 是否出现 organic search。
- 用户是否从首页进入后点击工具页。
- 工具页是否有页面浏览。
- calculator 禁用态是否造成快速离开。

已接入最小事件：

- `calculator_used`
- `filter_used`
- `item_search_used`
- `official_source_clicked`
- `related_tool_clicked`

### Clarity

重点看：

- 用户是否滚动。
- 是否点击工具入口。
- 是否在 value table 搜索。
- 是否卡在 calculator disabled state。
- 移动端有没有布局问题。

不要只看 session 数量，要看行为质量。
