# 07-广告变现计划

## 目标

在自然搜索流量稳定后，通过展示广告验证收入能力，并逐步优化到可规模化变现。

## 当前判断

检查日期：`2026-07-05`

当前 GA4 近 7 天：

| 指标 | 数值 |
| --- | ---: |
| activeUsers | 15 |
| sessions | 15 |
| views | 15 |
| source / medium | `(direct) / (none)` |

当前 Semrush 主词：

| Keyword | Volume | CPC | KD |
| --- | ---: | ---: | ---: |
| `steal a brainrot values` | 2,900 | $0.12 | 16 |
| `steal a brainrot trading values` | 4,400 | $0.03 | 19 |
| `sab values` | 3,600 | $0.00 | 19 |
| `best brainrot in steal a brainrot` | 8,100 | $0.02 | 27 |

结论：

- 当前阶段不应该接广告。
- 主词单独收入很低，即使排名很好也大概率只有个位数到十几美元/月。
- 真正的商业价值来自页面矩阵和 PV 扩张。
- 过早放广告会影响工具体验和索引验证。

## 广告收入公式

`月收入 = 月搜索量 * CTR * 每访客页数 * RPM / 1000`

目标：

`$3,000/月`

需要条件：

- 总关键词池月搜索量 500,000+。
- 月 PV 300,000-500,000+。
- RPM $6-$10。

## 变现阶段

### 阶段 1：不放广告

条件：

- 月 PV < 10,000。

当前所处阶段。

目标：

- 优先验证 SEO。
- 优先提高页面体验。
- 优先让核心页面收录。
- 不影响用户对数据可信度的判断。

### 阶段 2：轻量广告测试

条件：

- 月 PV 10,000+。
- GSC 有稳定点击。
- 核心工具页没有明显体验问题。

动作：

- 接入 AdSense 或同类平台。
- 每页 1-2 个广告位。
- 不遮挡工具。
- 不影响首屏主要内容。

关注：

- RPM。
- CLS。
- bounce rate。
- engagement time。
- calculator / table interaction 是否下降。

### 阶段 3：广告布局优化

条件：

- 月 PV 100,000+。

动作：

- 测试工具页下方广告。
- 测试列表中间广告。
- 测试移动端广告位置。
- 按页面类型分析 RPM。

禁止：

- 首屏满屏广告。
- 弹窗遮挡工具。
- 诱导误点。
- 影响未成年人体验的广告形式。

### 阶段 4：规模化收入

条件：

- 月 PV 300,000+。

动作：

- 评估 Ezoic / Mediavine Journey 等平台。
- 做广告位 A/B test。
- 按国家和页面类型优化。
- 建设更多高 RPM 页面。

## 广告位原则

优先位置：

- value table 下方。
- calculator 下方。
- updates 列表之间。
- item detail 相关内容区。
- FAQ 前后。

谨慎位置：

- calculator 工具内部。
- 首屏。
- 移动端顶部。

不要放：

- 按钮附近。
- 输入框旁边。
- 会导致误点击的位置。
- `TBD` 说明和 source 区域之间。

## Roblox / 儿童流量注意

这个站可能吸引年轻用户。

必须注意：

- 不收集不必要个人信息。
- 不做登录。
- 不做用户画像。
- 遵守广告平台政策。
- 广告内容要适合游戏受众。

如果后续加入评论、账号、投稿功能，需要重新审查隐私和合规。

## 开启广告前检查

必须满足：

- 月 PV 10,000+。
- 页面索引稳定。
- organic search 是主要来源之一。
- 没有核心布局问题。
- Core Web Vitals 没有明显异常。
- calculator / table / internal links 不被广告遮挡。

## 决策规则

如果广告收入很低：

- 不要加更多广告位。
- 优先提高 PV。
- 优先做关键词矩阵。

如果广告影响体验：

- 删除或下移广告位。
- 优先保留工具体验。

如果 RPM 高于预期：

- 分析高 RPM 页面类型。
- 扩展同类关键词。

## 验收标准

广告测试完成的标准：

- RPM 可追踪。
- 页面速度没有明显下降。
- CLS 没有明显恶化。
- 工具功能没有被遮挡。
- 用户行为没有明显变差。
- 广告不会削弱站点可信度。
