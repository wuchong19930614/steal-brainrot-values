## 主域名策略

主版本：

`https://www.stealbrainrotvalues.com/`

必须保持：

- 非 www 301 到 www。
- `NEXT_PUBLIC_SITE_URL=https://www.stealbrainrotvalues.com`。
- canonical 输出 www。
- sitemap URL 输出 www。
- robots 指向 www sitemap。
- GSC 提交 www sitemap。

不要出现：

- 页面最终访问是 www，但 canonical 是非 www。
- sitemap 和 canonical 使用不同域名。
- robots 指向另一套 sitemap。

## 当前 P0 清单

这是全套文档中唯一的当前优先级清单，其他文档只引用不复制。

1. 不再重复 request indexing；5 个核心页面已经收录。
2. 发布一个 `/trading-values/` 测试页：只呈现可核验的交易价值状态、来源和 `TBD` 原因，不伪造 value。
3. GA4 custom definitions 已可在报表中拆分；`item_search_used` 在输入停顿 600ms 后上报实际 `search_length`，并在 blur、Enter、Reset 或离开页面时补发尚未上报的快速搜索。
4. 首页 no-results 的 `Reset filters` 已有真实使用，且部署后没有新的该状态 dead click；保持现状，不再改首页或 calculator。
5. `/trading-values/` 部署后仅检查该新 URL 的状态、canonical、sitemap 和事件；继续观察 GSC 的 `firstIncompleteDate`，不据此批量扩页面。
6. 暂不发布 `/best-brainrots/`、`/all-brainrots/`、`/rarest-brainrots/` 或 item detail 页面，也不接广告。

## 完成标准

基础配置真正完成的标准：

- 5 个核心页面都不是 `URL is unknown to Google`。
- 首页和核心页面 canonical 均为 www。
- GSC user canonical 与 Google canonical 一致。
- sitemap submitted 5 个 URL 且无 errors。
- robots sitemap 与主 sitemap 一致。
- GA4 有真实 page_view。
- Clarity 有真实 session。
- 首页和核心工具页可被 Google 抓取。
