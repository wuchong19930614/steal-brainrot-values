## 主域名策略

主版本：

`https://www.stealbrainrotvalues.com/`

必须保持：

- 非 www 301 到 www。
- `NEXT_PUBLIC_SITE_URL=https://www.stealbrainrotvalues.com`。
- canonical 输出 www。
- sitemap URL 输出 www。
- robots 指向 www sitemap。
- GSC 提交 www sitemap。

不要出现：

- 页面最终访问是 www，但 canonical 是非 www。
- sitemap 和 canonical 使用不同域名。
- robots 指向另一套 sitemap。

## 当前 P0 清单

1. 在 GSC URL Inspection 中 request indexing：
   - `https://www.stealbrainrotvalues.com/calculator/`

3. 每天看 sitemap indexed 数是否从 0 增加。
4. 保持首页 title、canonical、robots、sitemap 稳定，不频繁改动。

## 每日检查

上线后前 7 天每天检查一次：

- 首页是否 200。
- sitemap 是否 200。
- robots 是否 200。
- 首页 URL Inspection coverage 是否仍是 indexed。
- 4 个工具页是否从 unknown 变成 discovered / crawled / indexed。
- GSC sitemap warnings / errors 是否仍为 0。
- GA4 是否有 page_view。
- Clarity 是否有 session。

如果发现 404、5xx、noindex、canonical 错误，先修复基础配置，不扩页面。

## 每周检查

每周记录：

- GSC indexed pages。
- GSC total clicks。
- GSC total impressions。
- GSC average position。
- GSC top queries。
- GA4 users / sessions / views。
- GA4 landing page。
- Clarity session 数和异常点击。
- Semrush 是否出现 organic keyword。
- Trends 是否继续上升或降温。

## 完成标准

基础配置真正完成的标准：

- 5 个核心页面都不是 `URL is unknown to Google`。
- 首页和核心页面 canonical 均为 www。
- GSC user canonical 与 Google canonical 一致。
- sitemap submitted 5 个 URL 且无 errors。
- robots sitemap 与主 sitemap 一致。
- GA4 有真实 page_view。
- Clarity 有真实 session。
- 首页和核心工具页可被 Google 抓取。
